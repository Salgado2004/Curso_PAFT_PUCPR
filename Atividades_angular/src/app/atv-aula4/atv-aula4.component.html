Exercício Aula 4

  • Crie uma pequena aplicação que permita adicionar itens a um carrinho
  • A aplicação deve ter uma tela de login, uma tela de carrinho com botões para adicionar produtos, finalizar a compra e fazer login.
  • Adicionar produtos pode ser feito a qualquer momento, mas finalizar compra só pode ser habilitado após o usuário fazer login
  • Para fazer login, basta apertar o botão de login, não há necessidade de fazer uma tela de login.
  • A aplicação deve ser dividida nos seguintes componentes:
    • Produto com descrição e valor
    • Item que possui um Produto e uma quantidade
    • Carrinho que contém uma lista de Produtos
    • Um serviço de login que simplesmente informa se o usuário fez login.
