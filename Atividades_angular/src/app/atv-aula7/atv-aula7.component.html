Exercício Aula 7

  1. Crie um pipe para converter um número em um valor monetário.
    • Exemplo: 123.45 é convertido para R$ 123,45
  2. Crie um pipe para filtrar todos os elementos pares ou ímpares de um array numérico.
  3. Crie um pipe para truncar um texto e mostrar reticências. Deve receber um parâmetro numérico
    que indica a quantidade máxima de caracteres.
    • Exemplo: Lasanha Quatro Queijos deve ser convertido para Lasanh ... caso o pipe receba o
      parâmetro 6. Mas Pipoca continua Pipoca pois tem apenas 6 caracteres.
